Nutrasheets.com

This website was developed around Nutrasheets, a food formulation and nutrition labeling add-on for Google Sheets. This website was built with Bootstrap and AngularJS on the front-end and Firebase Cloud Functions on the back-end to deliver the following functionality.

  • How to install and use Nutrasheets
  • Account management
  • Browse or suggest new nutrients for Nutrasheets

This website is in process to be released as a single page application with a lot of new features mid 2021.

Nutrasheets Add-on

This is a Google Sheets Add-on that creates a food formulation and nutrition labeling environment into Google Sheets. It uses Google Apps Script and Firebase on the back-end and Boostrap and AngularJS on the front-end to create a responsive, clean, highly functional user interfaces in HTML sidebars and modal dialogs in Google Sheets with the following features.

  • Create formulas and streams from 300,000+ ingredients
  • Create nutrition facts from formulas
  • Create ingredient statements from formulas
  • Create and store custom ingredients
  • Create batch sheets from formulas for the test kitchen or plant

Currently there are hundreds of users in academia, industry, and start-ups that use Nutrasheets to develop and label new products. There is a full pipeline of new features planned for launch in mid 2021. The long-term goal is for Nutrasheets to become a global powerhouse in the food formulation and nutrition labeling software space.
