Nutrasheets.com  

Nutrasheets Add-on

This is a Google Sheets Add-on that creates a food formulation and nutrition labeling environment into Google Sheets. It uses Google Apps Script and Firebase on the back-end and Boostrap and AngularJS on the front-end to create a responsive, clean, functional user interfaces in HTML sidebars and modal dialogs in Google Sheets with the following features.
  • Create formulas and streams from 300,000+ ingredients
  • Create nutrition facts from formulas
  • Create ingredient statements from formulas
  • Create and store custom ingredients
  • Create batch sheets from formulas for the test kitchen or plant

Currently there are hundreds of users in academia, industry, and start-ups that use Nutrasheets to develop and label new products. There is a full pipeline of new features planned for launch in mid 2021. Long-term I would like Nutrasheets to become a global powerhouse in the food formulation and nutrition labeling software space. Short-term I would like to secure funding to continue building out new features and growing my user base.
